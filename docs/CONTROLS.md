# CONTROLS — Hearthfolk Build 001

| Entrada | Acción |
|---|---|
| `WASD` | Desplazar cámara en el plano (relativo a rotación) |
| Botón central + arrastrar | Pan directo |
| Rueda del ratón | Zoom (12–80 m, exponencial) |
| `Q` / `E` | Rotar cámara |
| Doble clic en terreno | Centrar suavemente |
| `F` | Centrar en el asentamiento |
| `Espacio` | Pausa |
| `1` / `2` / `3` | Velocidad ×1 / ×2 / ×4 |
| `T` | Herramienta: marcar tala |
| `R` | Herramienta: zona residencial |
| `C` | Herramienta: demoler / cancelar |
| `I` | Herramienta: información |
| `Esc` | Cancelar herramienta activa |
| Clic izquierdo | Usar herramienta / seleccionar |
| Arrastrar (con herramienta) | Caja de selección / dibujar zona |
| Clic derecho | Cancelar herramienta |
| `F3` | Overlay de depuración |
| `F5` / `F9` | Guardar / Cargar |
